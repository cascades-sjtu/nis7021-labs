## 服务器验证方式

将文件名保存为你的学号，即替换命令中的 `student_id`，使用 `curl` 上传文件

- RISC-V 汇编

```bash
curl -F "file=@{student_id}.s" http://101.34.212.12:5000/upload/asm
```

- RISC-V 逆向

```bash
curl -F "file=@{student_id}.c" http://101.34.212.12:5000/upload/crypto
```
